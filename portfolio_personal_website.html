Responsive Design

Responsiveness

To create a more positive user experience I used media queries to make this website mobile friendly. Research shows that if the site is mobile friendly a user is more likely to stay on a website. With the ever increasing mobile use it's very important to support users that visit the website through mobile devices. Also being "searchable" is another big reason to have the website responsive, Google's SEO will show preference to websites that are designed to be mobile friendly.

Major highlights: Responsive design
